В рамках учебной практики было задание создать сайт магазин с использованием баз данных.
В качестве локального сервера был выбран Open Server. СУБД - phpMyAdmin.
Также, извиняюсь за выражение, накидал на скорую руку приложение на C# с использованием WPF.

Все файлы сайта хранятся в папке localhost.

Верстка сайта HTML + CSS. Главная стр. сайта index.php.
Информация о продукте/товаре берется из базы данных при помощи PHP.
Фотографии продукта хранятся в папке localhost\site\Arts, из базы данных берется путь к фото.

База данных - zakhar.xml

Приложение в директории "adding to database via WPF application in C#".
Функционал приложения скудный, только добавление нового продукта в базу. Сделано просто так, при надобности реализовал бы всё необходимое для работы с информацией о продукте.

Отчет по практике (содержит код, скриншоты внешнего вида сайта и приложения) - "отчет по практике.docx".
